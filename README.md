#Elite Schedule APP
#Project Information 
##Goal
learn how basics of creating a hybrid app with ionic

##Description
A small app following pluralsights tutorial on ionic that keeps track os basketball schedule for highschool team.
uses:
* caching
* promisses
* http requests
* services
* ui-router
